## Firebase 사용 현황

| 기능 | 서비스 | 경로 |
|------|--------|------|
| 로그인/회원가입 | Firebase Auth | - |
| 유저 정보 | Firestore | `Users/{uid}` |
| 프로필 사진 | Storage | `profile_images/{uid}.jpg` |
| 진단 이미지 | Storage | `diagnoses/{uid}/{timestamp}.jpg` |
| 진단 기록 | Firestore | `Diagnoses/{자동ID}` |

### Firestore Diagnoses 필드

| 필드 | 타입 | 내용 |
|------|------|------|
| userId | String | Firebase Auth uid |
| cropType | String | 작물명 |
| diseaseName | String | 병해명 (현재 더미 "노균병") |
| confidence | Double | 신뢰도 0.0~1.0 (현재 더미 0.92) |
| imageUrl | String | Storage 다운로드 URL |
| sickKey | String | NCPMS sickKey (예: "D00001166") |
| timestamp | Timestamp | 진단 시각 |
| isCounted | Boolean | 새 발생 인정 여부 (기본 false) |
| isHandled | Boolean | 처리 완료 여부 (기본 false) |
| memoTitle | String | 메모 제목 (기본 "") |
| memoContent | String | 메모 내용 (기본 "") |
| location | null | GPS 좌표 (미구현) |
| addressDo | null | 도 (미구현) |
| addressSi | null | 시 (미구현) |
